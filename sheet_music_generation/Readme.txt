In order to successfully generate sheet music, following things has to be done:

1. pip install music21 
2. Install Musescore https://musescore.org/en/download
2. Open music21_musescore_work_together_configurer.py in text editor
3. Change the path to the installed location of Musescore
4. run: python music21_musescore_work_together_configurer.py

You are all set !!
